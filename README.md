TrasBot
=======

Ficheros para el Trasbot, robot de uso educativo desarrollado por CrakerNano para Trastejant.es

![Trasbot](https://github.com/trastejant/TrasBot/blob/master/img/%C3%ADndice.jpeg)

## Componentes
- Arduino
- TrastShield (En desarrollo)
- Servo 9g
- Chasis impreso
- motoresde CC
